# Multi-Party Computation Signer Engine

This document describes the setup and prerequisites for the Multi-Party Computation (MPC) Signer Engine.
The MPC Signer Engine is designed to securely manage key shares and perform signing operations in a distributed manner.

## Compatibility

| OS                 | Status |
| ------------------ | ------ |
| macOS              | ✅     |
| Linux              | ✅     |
| Windows (via WSL2) | ✅     |
| Native Windows     | ✅     |

## Prerequisites

- [Docker](https://www.docker.com) and Docker Compose
- [Act](https://github.com/nektos/act) for local GitHub Actions testing
- [Rust](https://www.rust-lang.org) and Cargo
